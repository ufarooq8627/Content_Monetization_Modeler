

# 📊 YouTube Ad Revenue Prediction – Content Monetization Model

## 📌 Project Overview

This project builds an end-to-end **content monetization prediction system** using YouTube video analytics data.
The goal is to **predict ad revenue** based on reach, engagement, viewer retention, content category, device type, and geography.

The project follows a **complete data science lifecycle**:

* Data cleaning & validation
* Exploratory Data Analysis (EDA)
* Feature engineering
* Model training & evaluation
* Model interpretation & saving

---

## 🧠 Key Business Question

**What factors most strongly drive YouTube ad revenue, and how accurately can revenue be predicted from video performance metrics?**

---

## 🗂 Dataset

* Source: `youtube_ad_revenue_dataset.csv`
* Size: ~120,000 videos
* Key features:

  * Views, likes, comments
  * Watch time & video length
  * Subscribers
  * Category, device, country
  * Ad revenue (USD)

---

## 🧹 Data Preprocessing

* Removed logical errors (negative values, invalid watch time)
* Handled missing values
* Removed duplicate records
* Extracted date features (year, month, day of week)
* Applied **log transformation** to stabilize scale:

  * `log_views`
  * `log_subscribers`
  * `log_ad_revenue_usd`

---

## 🛠 Feature Engineering

Created engagement-focused features:

* **Engagement Rate**
  `(likes + comments) / views`
* **Watch Time per View**
  `watch_time_minutes / views`

These features decouple **quality of engagement** from raw scale.

---

## 📈 Exploratory Data Analysis (EDA)

### 🔍 Key Insights

* **Watch time per view** shows the strongest relationship with ad revenue
* Engagement rate has limited standalone impact
* Raw watch time is redundant with views
* Category, device, and country show **distributional shifts**, not strong standalone effects
* Log-transformed features produce more stable relationships

---

## 🧪 Modeling Approach

* Target variable: **`log_ad_revenue_usd`**
* Models evaluated:

  * Linear Regression
  * Ridge, Lasso, ElasticNet
  * Random Forest Regressor
* Pipeline includes:

  * Scaling numeric features
  * One-hot encoding categorical features

---

## 🏆 Model Performance

* **Random Forest** performed best
* Evaluation metrics:

  * R² Score
  * RMSE
  * MAE

---

## 🔎 Feature Importance (Random Forest)

Top drivers of ad revenue:

1. Watch time per view
2. Log views
3. Engagement rate
4. Subscriber count
5. Content category & country (secondary influence)

## 📌 Conclusion

Viewer retention (not just views) is the **primary driver of monetization**.
Optimizing **watch time per view** leads to more reliable revenue growth than focusing solely on reach.
